끝말잇기 게임

사용자가 첫 단어를 입력
컴퓨터가 해당 단어의 마지막 글자로 시작하는 새로운 단어를 제시
***만약 기존에 나왔던 단어라면 제시 불가***
제한시간은 10초 (설정 가능)
(단어를 입력하면 해당 단어의 뜻 가져오기)
난이도 조절 기능 추가(튬, 즙, '즘', '틱', '늄', '슘', '퓸', '늬', '뺌', '섯', '숍' 등 제거)


개발 과정
1. 한글 단어 전부 슬라이스로 import 
2. import한 슬라이스 중 명사만 걸러내어 새로운 슬라이스에 저장
3. 랜덤 모듈을 통해 슬라이스의 랜덤한 단어 출력



만들어야할 함수
1. main
2. 특정 단어로 시작하는 슬라이스을 반환하는 함수
3. 해당 단어가 이미 나왔는지 검사하고 불린값을 반환하는 함수


기타 기록 사항
1. 슬라이스에 append하기에는 성능이 부족. 파일에 작성한 뒤 파일을 문자열로 만드는 방식 채택.




1. 데이터베이스에서 단어 가져오기 
2. 데이터베이스에서 가져온 단어 편집하기 (한 글자 단어 제거)
3. 각자의 단어로 시작하는 파일명을 가진 파일을 생성
(3. 소켓 통신 적용)
4. 사용자가 단어 입력하면 해당 단어의 마지막 글자의 이름을 가진 파일 열기
5. 해당 파일에서 랜덤한 단어 반환
6. 사용자가 입력한 단어가 컴퓨터의 단어의 마지막 글자로 시작하는지 검사
7. 4~6 무한 반복
8. 10초가 지나면 탈락. 한 번 넘어갈 때마다 점수 10점 추가
9. 점수 기록 로그파일에 저장 && 게임 실행시 최고 기록 보여주기
